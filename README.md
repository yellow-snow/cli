# YellowSnow | CLI

## Goal

A CLI tool used to generate a powerful TypeScript framework project.

## Usage

### Compiling from source

```bash
git clone https://github.com/yellow-snow/cli
git checkout develop
npm install
npm run build
```

### Installation

```bash
npm install -g .
```

### Running

```bash
cd ~/
yc --new new-project
node ./new-project
```
